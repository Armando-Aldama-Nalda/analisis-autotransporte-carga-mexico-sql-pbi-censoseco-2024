# Análisis del Sector Autotransporte de Carga (México 2024)

## 📌 Objetivo del Proyecto
Analizar la estructura económica y la eficiencia operativa del sector autotransporte (SCIAN 484) mediante la segmentación por quintiles de producción y estratos de tamaño.

## 🛠️ Herramientas Utilizadas
* **SQL Server:** Limpieza de datos (ETL), creación de vistas y lógica de quintiles.
* **Power BI:** Modelado de datos y visualización de KPIs financieros.
* **Datasets:** Censos Económicos INEGI.

## 📈 Hallazgos Clave
* **Carga de Combustible:** Representa el 25% de los ingresos totales en empresas medianas.
* **Concentración:** El Quintil 5 (el más productivo) concentra el 77% de las empresas de mayor tamaño.
